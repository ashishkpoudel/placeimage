how to placeimage

Specify the image width and height https://use.placeimage.app/{width}x{height} and that's it Your image url should look like this https://use.placeimage.app/150x150

width and height

You have to specify width first. Height is optional. In absence of height image will be in square. https://use.placeimage.app/150 will produce an image of size 150x150

image format

You can have image in 4 different format [ jpg, png, gif and wepb ]. Just add available extension after image size. https://use.placeimage.app/300.jpg will have a resulting image in jpg format

custom text

adding text to placeimage is very easy and straight forward, just follow the given example: https://use.placeimage.app/300x100?text=custom text

background and font color

easily customize background and font color by using url in the format below https://use.placeimage.app/150/{background}/{foreground} https://use.placeimage.app/150/{background}/{foreground}?text=customtext
