Using placeimage.app

Specify the image width and height
https://placeimage.app/{{"{width}"}}x{{"{height}"}}
and that's it
Your image url should look like this https://placeimage.app/150x150.png

width and height

You have to specify width first. Height is optional. In absence of height image will be in square.
https://placeimage.app/150 will produce an image of size 150x150

image format

You can have image in 4 different format [ jpg, png, gif and wepb ]. Just add available extension after image size. https://placeimage.app/300.jpg will have a resulting image in jpg format

custom text

adding text to placeimage is very easy and straight forward, just follow the given example: https://via.placeholder.com/300x100?text=custom text
